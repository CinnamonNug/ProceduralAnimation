# ProceduralAnimation
An attempt to implement a [Jakobsen-Verlet](https://www.cs.cmu.edu/afs/cs/academic/class/15462-s13/www/lec_slides/Jakobsen.pdf) particle simulation and combine it with some simple kinematics in C++20.
